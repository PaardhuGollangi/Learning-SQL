# Basic Descriptive Statistics

## Counting

To count the number of rows present in the table :

```sql
SELECT COUNT(*) FROM store_sales;
```

To count number of rows present in each month :

```sql
SELECT
	month_of_year,
	COUNT(*)
FROM
	store_sales
GROUP BY
	month_of_year;
```
